meshed.util.func_name(func) → str[source]¶

The func.__name__ of a callable func, or makes and returns one if that fails. To make one, it calls unamed_func_name which produces incremental names to reduce the chances of clashing

meshed.util.name_of_obj(o) → Optional[str][source]¶

Tries to find the (or “a”) name for an object, even if __name__ doesn’t exist.

>>> name_of_obj(map)
'map'
>>> name_of_obj([1, 2, 3])
'list'
>>> name_of_obj(print)
'print'
>>> name_of_obj(lambda x: x)
'<lambda>'
>>> from functools import partial
>>> name_of_obj(partial(print, sep=','))
'print'
